# 项目说明

##* 项目基础
python 版本 ： python3

编译器 ： pycharm

***爬虫项目获取数据***

##* 项目文件介绍

### algorithm
算法文件

    TextClassifier对文本数据分类算法类

### model
实体文件

    TextRecorder文本数据处理类，管理文本数据


### view
视图文件

     App本项目视图类
   

### controller
控制器文件

    存放视图事件函数
    
### config
全局设置文件

     存放项目运行时全局变量参数，用于调控项目参数
